# Phase 5: Functional Features Included in the Solution

This document details the user-facing functional features developed for **RK Health**.

---

## 1. Key Application Features

### A. Patient Dashboard
- **Analytics Widgets**: Counts upcoming doctor consultations, active medicines, and logs.
- **Compliance Charting**: Visualizes active vs paused vs completed medications using Chart.js.
- **Daily timeline feed**: Chronological list of pills and timing scheduled for today.

### B. Scheduling Manager
- **Appointment forms**: Collects doctor info and notes, syncing them to Google Sheets.
- **Calendar invites**: Pushes events to Google Calendar.
- **Custom Filters**: Sorts appointments by name, doctor, and date ranges.

### C. Medication Reminders
- **Regimen Tracker**: Sets pill names, daily timestamps, and compliance status.
- **SMS notifications**: Sends text message alerts via Twilio.
- **SMS Manual Tester**: Enables instant reminder tests directly from the dashboard.

### D. Health Diary Logs
- **Observations list**: Logs symptom trackers, BP vitals, and side effects.
- **AI Summary Analyst**: Automatically translates terms and prints wellness guidelines.
