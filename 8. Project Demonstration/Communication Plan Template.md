# Phase 8: Communication & Team Plan

This document outlines the communication strategies and task involvement plan for presenting **RK Health**.

---

## 1. Stakeholders Communication Plan
- **Patients**: Educate on logging pill timings and requesting AI summaries.
- **Clinicians**: Show how to access logs to monitor patient vitals leading up to consultations.
- **Technical Reviewers**: Demo the Flask API architecture, local DB checks, and Google Apps Script bridge.

---

## 2. Team Involvement Matrix

| Role | Responsibility | Milestones |
| :--- | :--- | :--- |
| **Backend Engineer** | Flask routes, database schema, and server endpoints. | Milestones 1 & 2 |
| **Integration Developer** | Twilio SMS API and Google Sheets/Calendar bridge. | Milestone 3 |
| **UI/UX Designer** | Glassmorphic frontend portal and CSS styling. | Milestone 4 |
| **QA Specialist** | Unit testing and automated pytest suite. | Milestone 5 |
