Problem & Solution

Overview

Empowering Patient Compliance

Patients manage complex schedules across medical disciplines, leading to high non-adherence and missed critical details.

  • Medication schedules are managed via visual compliance badges.
  • Missed appointments are eliminated using SMS notification gates.
  • Complex diagnoses are translated using local LLM summarization.

Compliance Triggers

Twilio schedules reminders to alert phones at specified hour blocks daily.

Google Integrations

Updates are logged directly into Google spreadsheet database and calendars.

AI Translation Bridge

Aggregates records, identifies side effects, and outputs preventative guidelines.

System Architecture

Design Blueprint

Modular Service Design

RK Health splits responsibilities between local sandboxed servers and cloud microservices via an Apps Script Gateway.

  • Security first: Secrets are isolated in backend environment files.
  • Decoupled structure: Sheets database runs independently.
  • Fallback mechanics: The server falls back to SQLite offline modes if APIs fail.

Database & REST APIs

Backend Layer

Secure & Standard API contracts

Features standard HTTP responses and password encryption.

  • /api/auth: Password hashing and session management.
  • /api/appointments: Supports search query matching.
  • /api/medications: Regimen scheduler and phone validation.
  • /api/summary: Stats aggregator and AI text client.
// Local SQLite Schema
CREATE TABLE medications (
  id INTEGER PRIMARY KEY,
  user_id INTEGER REFERENCES users(id),
  medicine_name TEXT NOT NULL,
  dosage TEXT,
  frequency TEXT,
  reminder_time TEXT,
  status TEXT DEFAULT 'Active',
  phone_reminder INTEGER DEFAULT 0
);

Quick Execution

Deployment

Spin Up in Under 2 Minutes

Available on Docker and local environments.

  • Step 1: copy `.env.example` to `.env` and configure keys.
  • Step 2: Run `python backend/app.py` or compose.
  • Step 3: Visit `http://localhost:5000` to interact.
$ pip install -r requirements.txt
$ python backend/app.py
$ docker-compose up --build
